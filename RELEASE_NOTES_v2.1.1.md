# AMI v2.1.1 - Tray Icons Fix

**"Sai se sei davvero online." / "Know if you're really online."**

---

## English

Fix for tray bar status icons: symbols ✓, !, X now render correctly (no more squares/placeholders).

### What's Fixed
- Status icons drawn as vector shapes (no font dependency)
- Green: ✓ (online), Yellow: ! (unstable), Red: X (offline)

---

## Italiano

Fix per le icone di stato nella tray bar: i simboli ✓, !, X ora si vedono correttamente (niente più quadrati).

### Correzioni
- Icone disegnate come forme vettoriali (senza dipendenza da font)
- Verde: ✓ (online), Giallo: ! (instabile), Rosso: X (offline)
