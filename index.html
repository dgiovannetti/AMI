Funzionalità

🎯

Monitor in tempo reale

Ping multi-host in parallelo, verifica HTTP, soglie configurabili. Rileva online, instabile e offline.

🌓

Temi light / dark / auto

Interfaccia coerente con tema di sistema o scelta manuale. Dashboard e impostazioni sempre leggibili.

🔐

API con token opzionale

Endpoint /status, /health, /stats. Autenticazione Bearer opzionale per integrazioni sicure.

📊

Dashboard e grafici

StatCard, grafici latenza e stato, modalità compatta. Layout reattivo e accessibile.

🔔

Notifiche native

Toast Windows e notifiche macOS su cambio stato. Modalità silenziosa e preferenze per evento.

📝

Log CSV e OTA

Log eventi con rotazione. Aggiornamenti automatici da GitHub Releases con verifica checksum.
