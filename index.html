Quando scegliere AMI

  • Vuoi sapere se c’è internet usabile, non solo ping al router.
  • Lavori su reti instabili o con captive portal.
  • Preferisci un’app leggera in tray con dashboard e notifiche native.
  • Ti servono open source e privacy senza stack server.

Quando guardare altrove

  • PingPlotter — analisi multi-hop / route completa.
  • Zabbix / Nagios — monitoraggio enterprise su molti host e servizi.
  • PingInfoView — solo elenco ping minimal, footprint minimo senza GUI ricca.

Installazione

Download

Binari in GitHub Releases (ZIP windows / macos). Da sorgente, nella cartella 3.0:

pip install -r requirements.txt && PYTHONPATH=src python -m ami.main

Ultima release v3.1.4 Repository

macOS: scarica lo ZIP il cui nome contiene macos (es. AMI-v3.1.4-macos.zip), apri solo AMI.app dentro AMI-Package. Se Apple parla di Python, spesso si è aperto per sbaglio il file AMI in Mostra contenuto pacchetto → MacOS — non usarlo. Leggi MACOS_SECURITY.md e il file LEGGIMI_macOS.txt nello ZIP.
