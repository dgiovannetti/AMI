✨ Features

🎯

Real-time Monitoring

Multi-host ping testing with parallel execution. Tests connectivity against multiple hosts simultaneously for accurate status detection.

🌐

Smart Detection

Distinguishes between local network and internet connectivity. HTTP verification ensures actual web access beyond ICMP pings.

📊

Beautiful Dashboard

Dark modern interface with real-time graphs, latency trends, and comprehensive statistics. Professional design with animations.

🔔

Smart Notifications

Native Windows toast notifications on status changes. Configurable alerts with silent mode option for minimal interruptions.

📝

CSV Logging

Automatic event logging with timestamp, latency, and status. Exportable CSV format for analysis and troubleshooting.

⚙️

Fully Configurable

Customizable thresholds, polling intervals, test hosts, and notification preferences. Settings GUI with live apply.

🏆 Why AMI Stands Out

🔥

Real Internet Detection

Only AMI distinguishes LAN from Internet. Combines ICMP + HTTP verification. Other tools (PingPlotter, PingInfoView, MultiPing) use ICMP only.

💎

Modern UX

Dark neon PyQt6 interface. Competitors have dated 2000s UIs or technical dashboards. AMI is built for daily desktop use, not just troubleshooting.

⚡

Lightweight

~1% CPU, 50MB RAM idle. PingPlotter uses 200-300MB. Enterprise tools like Zabbix require 500MB-1GB server infrastructure.

🎯

All-in-One

Dashboard + logging + notifications + live settings. Competitors either lack features (PingInfoView has no graphs) or require complex setups.

✅

One-Click Portable

No installation, no server setup. Unlike Zabbix/Nagios (database + config) or PingPlotter (heavy installer + paid license).

🆓

Open Source

MIT License, fully transparent. PingPlotter and MultiPing are proprietary paid. AMI delivers enterprise-grade UX with community values.

AMI is the only lightweight desktop tool that combines real internet detection (HTTP + ICMP), modern UX, native notifications, and live CSV logging in one portable package.
