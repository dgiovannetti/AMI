# Archivio documentazione

Qui sono stati spostati **appunti storici / one-off** che non servono al flusso di sviluppo quotidiano, per tenere la root del repository più ordinata.

| File | Contenuto |
|------|-----------|
| `DMG_BUILD_SUMMARY.md` | Riepilogo build DMG (sessione di lavoro). |
| `RELEASE_FINAL_STATUS.md` | Stato finale pubblicazione release (checklist interna). |

Le note di rilascio versionate (`RELEASE_NOTES_v*.md`, `GITHUB_RELEASE_NOTES*.md`) restano in **root** per non rompere i link già pubblicati su GitHub.
