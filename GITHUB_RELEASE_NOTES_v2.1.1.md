# 🎉 AMI v2.1.1 - Tray Icons Fix

**"Sai se sei davvero online." / "Know if you're really online."**

Fix for tray bar icons: symbols ✓, !, X now render correctly (no more squares).

---

## English

### What's Fixed in v2.1.1

**Tray bar icons (✓, !, X)**
- Status icons now show correctly: checkmark (✓), exclamation (!), and X
- No more squares/placeholders - symbols are drawn as vector shapes (no font dependency)
- Green: ✓ (online), Yellow: ! (unstable), Red: X (offline)

---

## Italiano

### Correzioni in v2.1.1

**Icone nella tray bar (✓, !, X)**
- Le icone di stato ora si vedono correttamente: checkmark (✓), punto esclamativo (!) e X
- Niente più quadrati - i simboli sono disegnati come forme vettoriali (senza dipendenza da font)
- Verde: ✓ (online), Giallo: ! (instabile), Rosso: X (offline)
